About Me

My name is Ammar Badwan and I'm a full-stack software developer, specifically a MERN stack (MongoDB, ExpressJS, ReactJS and NodeJS) developer.

Completed six months Full Stack JavaScript web development course from Abdul Aziz AlGhurair School of Advanced Computing (ASAC) on August 2020, ranked top 20 in Jordan between software developers in number of commits made on GitHub in 2020.
